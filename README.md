# PelukApp

La PelukApp es un reproductor de audio local diseñado para crear y reproducir playlists personales.

## Características

<center>

Crear playlists propias. <br/>
Agregar canciones desde archivos de audio del dispositivo. <br/>
Reordenar y eliminar pistas dentro de una playlist. <br/>
Reproducir audio con controles de play/pausa, siguiente y anterior. <br/>
Mantener la última playlist y volumen usados entre sesiones. <br/>
Sin anuncios <br/>
Sin suscripciones <br/>
Interfaz de usuario de material, colores de acento y colores dinámicos (Android 12+) <br/>
y mucho más... <br/>

</center>
